ROSSANA DE ROSE — PROJECTS

PROJECT 1: RAG Aviation Safety Chatbot
Type: AI / NLP / RAG Application
Tech Stack: Python, FAISS, Sentence Transformers, Claude API (Anthropic), Streamlit
Description: A Retrieval-Augmented Generation (RAG) chatbot built on a curated knowledge base of aviation safety documents. The system divides documents into semantic chunks, creates vector embeddings using Sentence Transformers, stores them in a FAISS index, and retrieves the most relevant context to answer user questions. Claude (Haiku) generates accurate, context-grounded responses. The app is deployed with Streamlit and includes a source-citation panel showing which documents were used for each answer.
Key concepts demonstrated: RAG architecture, vector embeddings, semantic search, FAISS, LLM integration, Streamlit deployment.
Why it is unique: Rossana combined her 14 years of aviation expertise with her new AI skills to create a domain-specific knowledge assistant. This showcases both technical competence and real-world domain knowledge.

PROJECT 7: Ryanair Customer Analytics — Random Forest Classification
Type: Machine Learning / Customer Analytics
Tech Stack: Python, Pandas, Scikit-Learn, Random Forest, Streamlit, Matplotlib, Seaborn
Description: Rossana performed a full exploratory data analysis (EDA) on 2,200+ real Ryanair passenger reviews, built a Random Forest classification model achieving 95% accuracy and ROC-AUC 0.94 in predicting passenger recommendation likelihood, and deployed an interactive Streamlit dashboard with real-time ML inference. Key concepts demonstrated: end-to-end ML pipeline, EDA, feature engineering, Random Forest, model evaluation (accuracy, ROC-AUC), real-time inference, Streamlit deployment, business insight translation.
